Installation
==============

Voici comment installer **Envisage** :

1. Créez l’environnement conda :

   .. code-block:: bash

      conda env create -f environment.yml

2. Activez l’environnement :

   .. code-block:: bash

      conda activate env_isage

3. Installez le package localement :

   .. code-block:: bash

      pip install .
